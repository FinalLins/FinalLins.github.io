1. 概述

  在Java中，数组是一种引用类型（不是很清楚的同学可以看看我前面的博客），数组是用来存放同一种数据类型的集合，注意只能存放同一种数据类型（Object类型数组除外）。数组引用变量只是一个引用，数组对象和数组变量是分开存放的。举个例子：

    int[] array = new int[]{1, 2, 3, 4, 5};     //定义一个数组
    //声明的变量名就是array，它是存放到栈内存中，它存放到的是右边int[]对象的十六进制的地址
    //右边的int[]数组这个对象，它是存放到堆内存中

声明的变量名，也就是等号的左边，在C/C++中貌似叫左值。引用其实就是指针的概念，只是在Java中没有指针而已。


2. 数组的基本用法

2.1 数组的声明

  1. 数据类型[] 数组变量名 = new 数据类型[数组长度]。

     int[] array = new int[5];//定义一个长度为5的数组

  2. 数据类型 数组变量名[] = new 数据类型[数组长度]。

     int array[] = new int[5];//定义一个长度为5的数组

  3. 数据类型[] 数组变量名 = {元素1，元素2，元素3，元素4}。

     int[] array = {1, 2, 3, 4, 5};//定义一个长度为5的数组

第一种和第二种没啥区别，常用的是第一种，第二种方式看起来没那么直观。没仔细看还以为 array[] 是变量名

2.2 数组的使用

  初始化数组，当我们用关键字new进行操作的时候，就会在堆内存中申请一块内存空间存放该数组的对象，默认会根据数据类型对每个元素进行初始化赋值。

类型 说明 默认值
byte 基本类型 0
short 基本类型 0
char 基本类型 ‘\0000’(整型值为0)
int 基本类型 0
long 基本类型 0
float 基本类型 0.0F
double 基本类型 0.0D
boolean 基本类型 false
- 引用类型 null

下来我们直接看代码，进行初始化、访问其中某个元素和赋值等操作：

    int[] array = new int[5];       //定义一个长度为5的数组
                                    //该数组只能存放int类型
                                    //所有元素初始化值就int的默认初始化值0
    int element1 = array[0];        //访问数组第一个元素
    System.out.println(element1);   //输出值为0，验证了我们上面说过的数组初始化的时候会根据数据类型进行初始化
    array[1] = 2;
    System.out.println(array[1]);

    //遍历访问数组
    //数组下标范围 0 ~ (length - 1)
    int length = array.length;
    for (int i = 0; i < length; i++) {
        System.out.println(array[i]);
    }

当使用 0 ~ (length - 1) 范围外的下标访问数组时，系统就会抛出异常提示开发人员，数组越界咯。

java.lang.ArrayIndexOutOfBoundsException

3. 用数组实现对数据的增删改查

  任何一个数据结构的基本功能无非就是增、删、改、查，那么我们使用数组来实现以下：

public class MyArray {
    private static final int DEFAULT_CAPACITY = 20;
    private static final int ILLEGAL_INDEX = -1;
    private static final int INIT_VALUE = 0;

    /**
     * 定义一个数组
     */
    private int[] array;

    /**
     * 数组容量
     */
    private int capacity;

    /**
     * 元素真实个数
     */
    private int size;

    public MyArray() {
        this(DEFAULT_CAPACITY);
    }

    public MyArray(int capacity) {
        if (capacity <= 0) throw new IllegalArgumentException("数组容量必须大于0！");
        this.array = new int[capacity];
        this.capacity = capacity;
        this.size = 0;
    }

    /**
     * 向数组末尾添加一个元素
     *
     * @param element
     * @return
     */
    public boolean add(int element) {
        if (size == capacity) {
            return false;
        }
        array[size++] = element;
        return true;
    }

    /**
     * 删除某个元素
     *
     * @param element
     * @return
     */
    public boolean remove(int element) {
        int index = findElement(element);
        if (index == ILLEGAL_INDEX) {
            return false;
        }

        return removeByIndex(index);
    }

    /**
     * 通过索引删除元素
     *
     * @param index
     * @return
     */
    public boolean removeByIndex(int index) {
        if (index < 0 || index > size - 1) {
            return false;
        }

        if (index != size - 1) {
            for (int i = index; i < size - 1; i++) {
                array[i] = array[i + 1];
            }
        }
        array[size - 1] = INIT_VALUE;
        size--;
        return true;

    }

    /**
     * 通过索引获取对应的值
     *
     * @param index
     * @return
     */
    public int get(int index) {
        if (index < 0 || index > size - 1) {
            throw new IllegalArgumentException("数组越界！" + index);
        }
        return array[index];
    }

    public boolean modify(int oldElement, int newElement) {
        int index = findElement(oldElement);
        if (index == ILLEGAL_INDEX) {
            return false;
        }
        array[index] = newElement;
        return true;
    }

    public boolean modifyByIndex(int index, int newElement) {
        if (index < 0 || index > size - 1) {
            return false;
        }

        array[index] = newElement;
        return true;
    }

    /**
     * 根据元素查找索引位置（最简单的方式查找，也可以使用二分法查找）
     *
     * @param element
     * @return
     */
    private int findElement(int element) {
        for (int i = 0; i < size; i++) {
            if (array[i] == element) {
                return i;
            }
        }
        return ILLEGAL_INDEX;
    }

    /**
     * 打印数组
     */
    public void print() {
//        System.out.println(Arrays.toString(array));
        System.out.print("[ ");
        for (int i = 0; i < size; i++) {
            if (i == size - 1) {
                System.out.print(array[i]);
            } else {
                System.out.print(array[i] + ", ");
            }
        }
        System.out.println(" ]");
    }

}

代码还是蛮简单的，我们来测试下：

public static void main(String[] args) {
    MyArray myArray = new MyArray(10);              //默认容量20


    for (int i = 0; i < 5; i++) {
        myArray.add((int) (Math.random() * 100));   //随机生成数据添加到数据结构中
    }
    myArray.add(200);
    myArray.add(201);
    myArray.print();
    System.out.println("删除某个值 : " + myArray.remove(200));
    myArray.print();
    System.out.println("删除某个索引对应的值 : " + myArray.removeByIndex(1));
    myArray.print();
    System.out.println("修改某个值 : " + myArray.modify(201, 301));
    myArray.print();
    System.out.println("修改某个索引对应的值 : " + myArray.modifyByIndex(1, 302));
    myArray.print();
    System.out.println("显示某个索引对应的值 : " + myArray.get(1));
}

输出如下：(等号后面的是解释)

[ 92, 0, 4, 80, 12, 200, 201 ] ====> 源数据
删除某个值 : true ============> 删除200这个值
[ 92, 0, 4, 80, 12, 201 ]
删除某个索引对应的值 : true ====> 删除索引1的值
[ 92, 4, 80, 12, 201 ]
修改某个值 : true
[ 92, 4, 80, 12, 301 ]
修改某个索引对应的值 : true
[ 92, 302, 80, 12, 301 ]
显示某个索引对应的值 : 302

4. 特性

  • 末尾插入速度快。（指定位置插入速度依赖于查询和移动元素）
  • 查询速度慢。
  • 删除速度慢。
  • 数组创建后，长度是固定的。扩展性很差。
